# Terminal Commands

## **Terminal**

### What's in here?
```
ls (list)
```
### Where am I?
```
pwd (present working directory)
```
### Show all the commands you have run
```
history
```
### Clear output 
```
cmd k OR clear
```
### Search previous commands
```
CTRL+r
```

---
## **Directories**
### Change Directory
```
cd
```
### Create a new directory
```
mkdir (make directory)
```
---
## **Files**
### Create a file 
```
touch 
```
### Remove a file
```
rm (remove)
```
### Rename a file
```
mv (move)
```
